Windows 10 安装第三方主题

发表于 2019-09-15

为 Windows 安装第三方主题并非仅仅追求差异化，一款令人赏心悦目的主题不仅让你保持愉悦的工作状态，甚至可以保护视力。

强大的第三方主题配合插件可以迸发出无限的可能性。

为何折腾第三方主题

很多人对 Windows 主题的认识还停留在 Windows 7 那时的美化，抑或是 Windows 10 中的壁纸组轮换而已。

一款好的 Windows 主题应该拥有符合当今潮流的人性化的简洁 UI 设计，最好还具有一定的灵活度，可自行定制一些元素。

一款好的 Windows 主题还可以对很多软件进行渲染，实现整体视觉统一。

安装第三方主题

强烈建议在进行以下操作之前，先创建一个还原点。

下载主题

你可以在 这里 找到心仪的主题，也可以在 DeviantArt 搜索诸如 Windows, Theme, VisualStyle 之类的关键词。

以下内容以 cakeOS 为例

使用 UltraUXThemePatcher

下载最新版本

安装后重启，再次运行该软件。如果显示 Patched ，则为成功。

在 Windows 10 Creators Update 之后，这个软件可能无法获取所需文件的权限
C:\Windows\System32\uxinit.dll
C:\Windows\System32\uxtheme.dll
C:\Windows\System32\themeui.dll

目前唯一的解决方法：

以 Administrator 登录，右键需要获取权限的文件，切换到 “安全” 选项卡，点击 “高级”

在弹出的 “高级安全设置” 中，点击 “所有者” 后的 “更改”

在 “选择用户或组” 中，点击 “高级” ⇨ “立即查找”，找到你的账户

回到 “高级安全设置”，点击 “权限条目” 下的 “添加”，“选择主体”中选择你的账户后，“基本权限” 选中 “完全控制”

此时你的账户就已获取所需权限，照此获取权限
uxinit.dll, uxtheme.dll, themeui.dll

安装主题

将下载的主题文件放入 C:\Windows\Resources\Themes

虽然 cakeOS 不区分 Windows 10 的版本，但有些主题需要根据 Windows 版本放入不同文件。你可以按下 Win+R 输入 winver 查看 Windows 版本。

正确放入文件后就可以在 “设置” ⇨ “个性化” ⇨ “主题” 中选择第三方主题并应用。

对于暗色主题，Windows 10 默认会加载
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\DefaultColors

备份该注册表值，取得权限并删除『DefaultColors』

cakeOS SUITE

cakeOS 还提供了与主题配套的图标，鼠标光标和壁纸。

  • Icons

下载 cakeOS colors iPACKS ，解压后将 .ee 改为 .exe ，然后安装即可

  • Cursors

下载 cakeOS Cursors ，双击 Install.inf 安装，然后在 “设置” ⇨ “个性化” ⇨ “主题” ⇨ “鼠标光标” 中启用

  • Rainmeter by Eldog-02

你还可选用 Rainmeter 以达到更好的视觉效果
